0.1.1

Wednesday 13th June

  • It’s now possible to copy and paste values between old versions and the current version in the versions browser.
  • Fixed a date formatting bug were dates were saved in the 12 hour format rather than the 24 hour format.
  • Increased the clickable area for the show appcast settings disclosure triangle.
  • Fixed a bug were Appcastr would read the publication date of the appcast file if there was no publication date specified for the update.
  • When resizing the main window to display the appcast settings, the appcast settings now appear before the window resizes, rather than suddenly appearing when the window finishes resizing.
  • Added information to the about screen.
